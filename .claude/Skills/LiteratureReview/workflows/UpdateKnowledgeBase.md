# Update Knowledge Base Workflow

## Purpose
Save paper analysis and synthesized knowledge to History/Papers/ and maintain searchable index.

## Usage
```
Save paper analysis to History/Papers/ and update index
```

## Input
- Paper analysis results
- Synthesis or comparison outputs
- Extracted methods or protocols
- Citation information

## Process
1. Structure analysis data for storage
2. Generate unique paper identifier/filename
3. Save analysis to History/Papers/ directory
4. Extract key metadata and tags
5. Update papers index file
6. Link related papers and concepts
7. Update search/retrieval index

## Output
- Saved paper analysis file
- Updated index with new entry
- Cross-references established
- Knowledge base ready for search

## Related Workflows
- AnalyzePaper.md - Generate analysis to save
- CompareStudies.md - Save synthesis results
- ExtractMethods.md - Save extracted protocols
