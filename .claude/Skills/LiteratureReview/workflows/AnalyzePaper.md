# Analyze Paper Workflow

## Purpose
Fetch and analyze paper metadata using the AnalyzePaper.ts tool.

## Usage
```
Use AnalyzePaper.ts to retrieve metadata and analyze a scientific paper
```

## Input
- Paper identifier (DOI, PubMed ID, or arXiv ID)
- Analysis depth (quick, standard, comprehensive)

## Process
1. Fetch paper metadata from source
2. Extract key information (title, authors, abstract, methods)
3. Analyze contribution and significance
4. Identify key findings and limitations

## Output
- Structured paper analysis
- Key takeaways and insights
- Relevance assessment

## Related Workflows
- UpdateKnowledgeBase.md - Save analysis results
- CompareStudies.md - Compare with other papers
