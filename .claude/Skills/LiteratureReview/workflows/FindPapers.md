# Find Papers Workflow

## Purpose
Search for relevant scientific papers using SearchPapers.ts across PubMed and arXiv databases.

## Usage
```
Use SearchPapers.ts to search PubMed/arXiv for papers matching criteria
```

## Input
- Search query or keywords
- Database selection (PubMed, arXiv, or both)
- Date range filters
- Maximum number of results

## Process
1. Construct search query with appropriate filters
2. Execute search across selected databases
3. Retrieve paper metadata and abstracts
4. Filter and rank results by relevance
5. Present curated list of papers

## Output
- List of relevant papers with metadata
- Quick summaries of each paper
- Download/access links
