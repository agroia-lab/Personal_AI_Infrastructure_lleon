# Compare Studies Workflow

## Purpose
Synthesize and compare multiple papers using ComparePapers.ts to identify patterns and contradictions.

## Usage
```
Use ComparePapers.ts to synthesize multiple papers and find connections
```

## Input
- List of paper identifiers or analysis files
- Comparison criteria (methods, results, conclusions)
- Synthesis focus area

## Process
1. Load analysis data for multiple papers
2. Extract comparable elements from each study
3. Identify commonalities and differences
4. Synthesize findings across studies
5. Highlight contradictions or gaps
6. Generate comparative summary

## Output
- Comparative analysis table
- Synthesis of findings
- Areas of consensus and disagreement
- Research gaps identified
